# 📘 Exercícios de JavaScript — Estruturas de Repetição

Este repositório contém a resolução de uma série de exercícios em JavaScript com foco no uso de **estruturas de repetição**, especialmente o `while`, para controle de fluxo e validação de dados.

---

## 🎯 Objetivo

Aplicar conceitos fundamentais de lógica de programação utilizando JavaScript, incluindo:

* Validação de entrada de dados
* Controle de repetição com `while`
* Interação com o usuário via `prompt` e `alert`
* Organização e versionamento de código com Git

---

## ✅ Resultados Alcançados

* ✔️ Programas desenvolvidos em JavaScript (`.js`)
* ✔️ Todos os exercícios testados e funcionando corretamente
* ✔️ Uso consistente de estruturas de repetição
* ✔️ Versionamento realizado em repositório local e remoto com Git

---

## 🧠 Exercícios Resolvidos

### 1️⃣ Número positivo

Solicita ao usuário um número e continua pedindo até que seja digitado um valor positivo válido.

---

### 2️⃣ Média de notas

Permite ao usuário inserir várias notas (de 0 a 10).
O programa encerra ao digitar um número negativo e exibe a média das notas.

---

### 3️⃣ Total de compras

Recebe valores de compras continuamente.
Quando o usuário digita `0`, o programa exibe o valor total acumulado.

---

### 4️⃣ Nome completo com validação

Solicita o primeiro nome e o sobrenome separadamente.
Caso algum campo esteja vazio, o programa pede novamente até o preenchimento correto.

---

### 5️⃣ Simulação de saque bancário

Simula um sistema de saque com saldo inicial definido.

* Verifica se há saldo suficiente
* Solicita novo valor caso o saque seja inválido
* Encerra quando o saldo chega a zero ou negativo

---
